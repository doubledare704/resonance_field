# Preflight Hook

Use this before starting work on Resonance Field.

## Check

- Read `initial.md`.
- Inspect the current tree and confirm whether the task is client, backend, bridge, or docs.
- Identify the smallest file set that can satisfy the request.
- Look for existing message types, node rules, and daily reset logic before changing anything.

## Output

- State the impacted layer.
- State the files you expect to touch.
- State any risk that could change behavior across the client/backend bridge.
